# skills
Skills that can be used by Bots to make life easier

## General Openclaw Skills

### Tokenuse
Use skill ```/session_status``` to get the token use for a bot. No created skill is necessary

### Learn
Use skill ```/learn``` to add permanent instructions to your bot

### Cleanup
Use skill ```/cleanup``` to go through local files, compact them, clean duplicate work

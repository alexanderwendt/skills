# Task Planning

Use this skill after a story or design is approved.

## Task Plan Format

```markdown
## Goal

<Approved story/design goal.>

## Tasks

1. <task>
   Owner: <agent>
   Depends on: <dependency or None>
   Verification: <check>

## Risks

- <risk>

## Done When

- <observable completion condition>
```

Keep tasks small, ordered, and verifiable. Do not add scope not present in the approved story or design.

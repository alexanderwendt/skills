# Architecture Decisions

Record significant decisions in `docs/architecture/decisions.md`.

## Decision Format

```markdown
## ADR-000: <Title>

Status: Proposed | Accepted | Superseded
Date: YYYY-MM-DD

### Context

<Why this decision is needed.>

### Decision

<What was decided.>

### Consequences

- <positive or negative consequence>
```

Prefer decisions that are reversible unless requirements demand otherwise.
